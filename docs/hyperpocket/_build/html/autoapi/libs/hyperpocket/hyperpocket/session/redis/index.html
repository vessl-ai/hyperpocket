  • auth_resolve_uid (str) – a UID used to verify whether the user has completed the authentication process. if set this value as None, it’s regarded as active session

  • auth_context (Optional[AuthContext]) – authentication context. in pending session, this value is None. in active session this value shouldn’t be None
